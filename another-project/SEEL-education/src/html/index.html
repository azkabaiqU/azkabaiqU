Website yang bernama Seel yang dibuat oleh 3 anggota tim, website ini tentang edukasi pembelajaran, website ini berisi 2 page, dibuat tanpa tailwind karena ini dibuat sebelum saya belajar tailwind, atau baru pertama kali belajar tailwind, saya sebagai developer utama, membuat website ini dengan bantuan 2 anggota tim lainnya, yaitu Reveeno dan Mawaldy, website ini dibuat untuk mengikuti lomba yang diadakan oleh Universitas Negeri Malang, website ini dibuat dengan menggunakan HTML, CSS, dan JavaScript, website ini juga sudah responsive.
